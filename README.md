<h1> Sistema de Lavação - PHP </h1>

<h3> Neste repositório apresentamos o <b>Sistema Norteador</b>, que consiste no <b>Sistema de Lavação</b> </h3>

<p> Este programa é uma aplicação web voltada para uma empresa de lavação de carros, constituindo-se das seguintes funcionalidades:</p>
<ul>
  <li> Cadastros:
    <ul> 
      <li> Cliente </li>
      <li> Veículo </li>
    </ul>
  </li> 
  <li> Login:
    <ul>
      <li> Cliente </li>
      <li> Veículo </li>
    </ul>
  </li>
  <li> Contatos:
    <ul>
      <li> Empresa </li>
    </ul>
  </li>  
</ul>
<p> Para este projeto foram utilizadas as tecnologias e linguagens: </p>
<ul>
  <li> HTML, CSS e JavaScript (frontend) </li>
  <li> PHP (backend) </li>
  <li> MySQL (banco de dados) </li>
</ul>
